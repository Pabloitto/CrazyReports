Base URL

All requests to Crazy Reports must be submitted to the base URL. We provides you with an option of a response as a JSON object.

HTTP Methods

All requests are submitted through the HTTP POST or GET method using UTF-8 encoding and URL encoded values. Expected "Content-Type" for POST is "application/json" when posting parameters as a JSON object.
